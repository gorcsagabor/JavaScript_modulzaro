1. Mikor használandó a JS egy weboldal felépítése során, hogyan használható?
JS egy programozási nyelv, amelyet a weboldalak interaktivitásának és dinamizmusának hozzáadására használunk. A JS kód segítségével kezelhetjük a felhasználói eseményeket, manipulálhatjuk az adatokat, kommunikálhatunk API-kkal, létrehozhatunk animációkat és rengeteg egyébb lehetőség áll rendelkezésünkre. A JS kód használatához megkell hívnunk a kódokat, hogy lefussanak. Meghívási lehetőségek: Beágyazott JavaScript kód, Külső JavaScript fájl, egy külön JavaScript fájl, amely tartalmazza a kívánt kódot eseménykezelők attribútumon keresztül. Eseménykezelők attribútumon keresztül: Egyes HTML elemekhez közvetlenül hozzáadhatunk eseménykezelők kódot az attribútumokon keresztül

2. Sorold fel, hogyan határozhatunk meg egy értéket és röviden definiáld őket.
JavaScript-ben az értékek meghatározására változókat használunk. A változók a következőképpen definiálhatók röviden: VAR, LET, CONST, Ezek a kulcsszavak lehetővé teszik az értékek tárolását változókban.

4.Hogyan deklarálhatunk egy változót, és hogyan alkalmazhatóak?
Először meg kell adnunk a változó típusát (VAR, LET, CONST), változó nevét. Ha szükséges, kezdőértéket is adhatunk a változónak. A változók hatásköre és láthatósága attól függ, hogy hol vannak deklarálva.

5.Mikor kell elágazásokat használni? Ismertesd őket (2-3 mondatban).
JavaScript-ben akkor használunk, amikor szeretnénk különböző feltételek alapján irányítani a program végrehajtását. Az elágazások lehetővé teszik, hogy a program döntsön arról, hogy melyik kódrészletet hajtsa végre, a feltételek alapján.

9.Hogyan vizsgálható egy tömb? 2-3 példát írj kérlek.
Tömb hossza: A tömböknek van egy length tulajdonságuk, amely visszaadja a tömbben található elemek számát. Tömb ürességének ellenőrzése: A tömbök ürességét az length tulajdonság segítségével lehet ellenőrizni. Adott elem jelenléte a tömbben. Tömb típusának ellenőrzése.

10.Hogyan mehetünk végig egy tömb/object/lista hosszán?
JavaScript-ben többféle módon tudsz végigmenni egy tömbön, objektumon vagy listán, attól függően, hogy milyen adatszerkezetről van szó.

11.Sorolj fel néhány events metódus.
Az addEventListener metódus segítségével hozzáadhatunk egy eseménykezelőt egy adott eseményhez egy elemhez. Az onscroll eseményt használhatjuk egy elem görgetése eseményének kezelésére Az onclick esemény metódust használhatjuk egy elemhez való kattintás eseményének kezelésére.

12.Miért érdemes függvényeket használni?
Újrafelhasználhatóság, Kód rendezettsége: Függvények segítségével rendezheted a kódot logikus egységekre. Kód újratervezése és karbantarthatóság, Hibaellenőrzés és hibakezelés.

13.Hogyan paraméterezhetünk fel egy függvényt?
A paraméterek lehetővé teszik, hogy adatokat adjunk át a függvénynek, amelyeket a függvény a végrehajtás során felhasználhat. A paraméterek definiálásához a függvény deklarációjában, a nyitó zárójel előtt kell felsorolni őket.

14.Mi az a function visszatérési érték?
Érték visszaadása egy függvény segítségével eredményeket vagy számított értékeket adhatsz vissza.

15.Mikor futtatható egy függvény?
Egy függvény futtatható, amikor azt meghívjuk. A függvény meghívása azt jelenti, hogy végrehajtjuk a függvényben található kódot, és a függvény műveleteit elvégezzük.

16.Mi az a DOM?
Egy programozási interfész, amely a weboldalak strukturált reprezentációját nyújtja. A DOM modell lehetővé teszi a JavaScript számára, hogy manipulálja és módosítsa a weboldal tartalmát és stílusát.

17.Mire utasít a következő példa: document.getElementByClassName(“example”).innerText
Ez a kód megpróbálja megtalálni az összes "example" osztályú elemet a dokumentumban, majd visszaadja a kiválasztott elemek szövegtartalmát.

18.Hogyan nyerhető ki egy form input mező értéke és hogyan jeleníted azt meg HTML-ben?
Az id attribútum segítségével azonosítjuk az input mezőt és a gombot. Az onclick eseménykezelőhöz rendeljük a függvényt, amely meghívódik, amikor a gombra kattintunk.

19.Mi az a callback function?
Egy callback function (visszahívási függvény) egy olyan függvény, amelyet egy másik függvénynek vagy metódusnak adhatunk át paraméterként.

20.Definiált: Date();
Az Date() egy beépített objektum a JavaScriptben, amelyet a dátum és idő kezelésére használhatunk.

21.Definiáld: onclick();
Az onclick() egy eseménykezelő metódus a JavaScriptben, amelyet gyakran használunk az interaktív weboldalakon

22.Definiáld: function Click(){…};
A function Click() {...} egy függvény definíció. A Click nevű függvényt létrehozza, és a kapcsos zárójelek közötti blokkban tartalmazza a függvény testét, vagyis azt a kódot, amit a függvény végrehajt, amikor meghívják.
